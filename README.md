# Panda - Customer Support Ticketing UI

Panda is a modern, responsive customer support ticketing dashboard designed to help support teams manage and track customer inquiries efficiently. This project demonstrates a clean, professional user interface with a focus on usability and aesthetics.

## Features

*   **Responsive Dashboard:** A comprehensive overview of ticket statistics, including active, resolved, and urgent tickets.
*   **Ticket Management:** A detailed table view to track ticket status, customer details, and updates.
*   **Reports Generation:** A dedicated section for creating new support tickets with categorization and assignment options.
*   **Settings & Configuration:** A settings page to manage application preferences, notifications, security, and team members.
*   **User Profile:** A profile section displaying user information and performance statistics.
*   **Dark/Light Mode:** Built-in theme toggler to switch between dark and light visual themes.
*   **Responsive Design:** Fully optimized for desktop, tablet, and mobile devices with a collapsible sidebar.

## Technologies Used

*   **HTML5:** Semantic markup for structure.
*   **CSS3:** Custom styling with CSS variables for theming and Flexbox/Grid for layout.
*   **JavaScript:** Logic for navigation, theme toggling, and sidebar interactivity.
*   **Remixicon:** Open-source icon library for UI elements.
*   **Google Fonts:** Typography using Lato and Montserrat.

## Getting Started

To view the project locally:

1.  Clone the repository:
    ```bash
    git clone https://github.com/BrightDev10-Cloud/Panda.git
    ```
2.  Navigate to the project directory:
    ```bash
    cd Panda
    ```
3.  Open `index.html` in your preferred web browser.

## Project Structure

*   `index.html`: Main HTML file containing the application structure and views.
*   `style.css`: Global styles, layout definitions, and theme variables.
*   `index.js`: JavaScript file handling UI interactions and logic.
*   `assets/`: Directory containing images and icons.
